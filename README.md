# Watcha Clone
*구글 설문지와 유사한 설문 관련 주제의 프로젝트입니다.
*관리자가 설문지를 CRUD 할 수 있습니다.
*설문 참여자가 있을 경우 설문 참여내역 및 레포트 확인이 가능합니다.
*구글 설문지처럼 설문을 배포, 참여자는 이름 확인만으로 설문 참여가 가능합니다.
